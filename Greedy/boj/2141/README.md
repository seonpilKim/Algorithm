# [[2141] 우체국](https://www.acmicpc.net/problem/2141)

## 🤔접근
- 왼쪽 마을의 사람 수와 오른쪽 마을의 사람 수가 비슷한 마을에 우체국을 세우는 것이 최적이다.
___
## 💡풀이
- <b>알고리즘 & 자료구조</b>
	- `Greedy`
- <b>구현</b>
	- 입력 받으면서 총 마을 사람 수를 더해주었다.
	- 먼저 위치 순으로 정렬해주고, 차례대로 탐색하면서 `현재까지의 마을 사람 수 >= (총 마을 사람 수 + 1) / 2` 인 경우, 바로 출력해주었다.
		- 마을 사람 수가 홀수인 경우를 생각하여 1을 더해준 값을 나누어야 한다.
